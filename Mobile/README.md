# Tech Assassin (Android)

This is the mobile application for the Tech Assassin community, built with React Native (Expo).

## Features
- **Premium Cyberpunk UI**: Dark-themed, high-performance interface with red accents.
- **Micro-Animations**: Powered by `react-native-reanimated` for a smooth native feel.
- **Custom Typography**: Uses Space Grotesk and Inter fonts to match the web app.
- **Android Optimized**: Edge-to-edge layout, custom status bar, and Material 3 inspired components.

## Getting Started

### Prerequisites
- Node.js
- Expo Go (on your Android device) OR an Android Emulator

### Installation
1. Navigate to the mobile directory:
   ```bash
   cd Mobile
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App
- To start the development server:
  ```bash
   npx expo start
   ```
- To run directly on an Android emulator or connected device:
  ```bash
  npx expo run:android
  ```

## Tech Stack
- **Framework**: React Native (Expo SDK 52+)
- **Styling**: React Native StyleSheet (Tailwind-ready colors)
- **Icons**: Lucide React Native
- **Animations**: React Native Reanimated
- **Fonts**: Expo Google Fonts
